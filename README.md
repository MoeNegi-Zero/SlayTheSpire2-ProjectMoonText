This is a Repository which contains a godotscript can help you to spawn Text in ProjectMoonStyle  
You can simply install this as mod but make sure you have follow the guide below  
If you install this as mod, you need to create a JSON file which name should be "MoonTextOption.json" in you mod path => (Mods/mods)/MoonText/MoonTextOption.json
Please ensure that the format in your JSON file is consistent with the sample file

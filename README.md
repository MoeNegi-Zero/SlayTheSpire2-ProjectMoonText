This is a Repository which contains a godotscript can help you to spawn Text in ProjectMoonStyle  
  
It will check BBcode to help you color Text in a sample way  
  
You can simply install this as mod but make sure you have follow the guide below:  
  
If you install this as mod  
You need to create a JSON file which name should be "MoonTextOption.json" in you mod path => (Mods/mods)/MoonText/MoonTextOption.json  
Please ensure that the format in your JSON file is consistent with the sample file  
  
If you just wanna randomly spawn some MoonStyleText  
  
Here is a sample 
  
{ "text":"[color=gold]This is a Test！[/color]"}
  
If you just wanna spawn MoonStyleText like the lyrics in BossBattle  
  
Here is a sample  
  
{ "text":"[color=gold]This will appear when you enter battle 5s later！[/color]", "time": 5.0},  
{ "text":"[color=gold]This will appear when you enter battle 10s later！[/color]", "time": 10.0}  

Have fun :D
